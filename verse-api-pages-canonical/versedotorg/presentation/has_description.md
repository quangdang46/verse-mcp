# has_description interface
Learn technical details about the has_description interface.
Interface that provides descriptive names or text.
|
---|---

## Members
This interface has data members, but no functions.
### Data
Data Member Name | Type | Description
---|---|---
`Name` | `?message` |  Name.
`Description` | `?message` |  Full description.
`ShortDescription` | `?message` |  Short description with only the essential information.
