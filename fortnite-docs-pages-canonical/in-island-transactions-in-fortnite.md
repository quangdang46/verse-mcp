# In-Island Transactions
Generate a revenue stream with your own items by using In-Island Transactions
![In-Island Transactions](https://dev.epicgames.com/community/api/documentation/image/a5b77237-a256-495a-82fe-dc48a31ebee4?resizing_type=fill&width=1920&height=335)
**In-Island Transactions** provides a way for you to create your own in-game economy by offering items to players in exchange for V-Bucks. Use Verse to define the item name, description, and price (in V-Bucks). Review your catalog of offers and monitor sales performance from within [Creator Portal](https://dev.epicgames.com/documentation/en-us/fortnite/using-creator-portal-in-fortnite-creative) while the system ensures a central, unchangeable record of every transaction.
Setting Up An In-Island Transaction
Explore In-Island Transactions Feature Example to learn more about the code in UEFN. You can also create a unique experience for your players by learning more about the In-Island Transactions device and how you can build your own custom shopfront UI with the In-Island Transactions [User Interface example](https://dev.epicgames.com/documentation/en-us/fortnite/user-interfaces-feature-template-in-unreal-editor-for-fortnite).

The **[Engagement Payout Program Terms](https://legal.epicgames.com/fortnite/developer-terms)** have been updated to the **[Fortnite Developer Terms](https://legal.epicgames.com/fortnite/developer-rules)** and includes provisions addressing In-Island Transactions.
