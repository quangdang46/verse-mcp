# Mobile Development
Reach mobile gamers with islands designed and optimized for handheld devices. 
![Mobile Development](https://dev.epicgames.com/community/api/documentation/image/52c2904c-da88-4dab-860b-0cd79d2a3fcb?resizing_type=fill&width=1920&height=335)
Developing for mobile in Unreal Editor for Fortnite (UEFN) gives you the opportunity to tap into Fortnite’s massive, cross-platform player base. Access a scalable path that countless indie developers and studios use to create visually striking mobile games that benefit from Fortnite’s infrastructure and discovery system.
The pages in this section give you the resources to design, test, and optimize islands for mobile play. With mobile hardware in mind, accommodating smaller screens, lower power budgets, different aspect ratios, and touch interfaces becomes a priority, so get ready to think mobile-first!
