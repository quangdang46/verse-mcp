# Realistic Assets, Characters, and Environments
Elevate your games to new heights with realistic visuals that captivate players. 
![Realistic Assets, Characters, and Environments](https://dev.epicgames.com/community/api/documentation/image/a48c62de-2554-4a13-bd1e-2fe4929bf764?resizing_type=fill&width=1920&height=335)
On this page
Using Unreal Editor for Fortnite (UEFN) to add realistic assets, characters, and environments to your games lets you leverage powerful Unreal Engine capabilities inside the Fortnite ecosystem. UEFN provides advanced tools for creating and working with high-fidelity models, detailed textures, and dynamic environments, so each element of the game world is immersive and lifelike.
One of the significant benefits of using UEFN is its optimization for cross-platform performance. With the right optimization, high-quality assets can run smoothly on all platforms that support Fortnite, from high-end PCs to mobile devices. This ensures a consistent and visually stunning experience for players, regardless of their device.
