# Your First Island: Level Up with Verse
You've built your first island. Ready to take it to the next level? 
![Your First Island: Level Up with Verse](https://dev.epicgames.com/community/api/documentation/image/abf00ef3-3cd4-4069-bfb8-04828e7b3a35?resizing_type=fill&width=1920&height=335)
######  Prerequisite topics 
In order to understand and use the content on this page, make sure you are familiar with the following topics:
  * [Build Your First Island in Fortnite](https://dev.epicgames.com/documentation/en-us/fortnite/build-your-first-island-in-fortnite)


If you've completed the [Build Your First Island in Fortnite](https://dev.epicgames.com/documentation/en-us/fortnite/build-your-first-island-in-fortnite) tutorial, sweet! This new tutorial picks up right where it left off!
If you haven't completed the **Build Your First Island** tutorial, you can still jump in and get started by creating a new island with the **My First Island** template under **Feature Examples** , and then start this tutorial. 
This time around, you'll learn how to: 
  1. Create a **combo system** that rewards skilled players with extra points.
  2. Add a **timer** to keep the pressure on and end the game.
  3. Create a **bonus target** to surprise players with a chance to earn extra time.
  4. Keep things fresh with **new weapons** on each completed combo.
  5. **Polish your project** as you learn more Verse!
