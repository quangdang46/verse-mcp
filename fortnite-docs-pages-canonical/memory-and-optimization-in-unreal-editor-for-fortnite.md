# Memory and Optimization
Optimize your project to overcome memory limits and improve your game's performance on all supported platforms. 
![Memory and Optimization](https://dev.epicgames.com/community/api/documentation/image/607148ae-13f5-4e52-85c0-b4e67975bac3?resizing_type=fill&width=1920&height=335)
On this page
Unreal Editor for Fortnite (UEFN) gives you industry-leading tools and access to a large, thriving community of players eager to try out your creations. In order to make your experiences available on all Fortnite-supported platforms, memory and project size limits were put in place.
Whether you’re trying to optimize your island to improve memory usage, enhance your island’s performance, or get more assets into your project without hitting the size limit, this section contains helpful tips and tricks to guide you in the right direction.
Check out [Import Content and Islands](https://dev.epicgames.com/documentation/en-us/fortnite/import-content-and-islands-in-unreal-editor-for-fortnite) section for more information on configuration and memory considerations when getting an asset ready for import.
