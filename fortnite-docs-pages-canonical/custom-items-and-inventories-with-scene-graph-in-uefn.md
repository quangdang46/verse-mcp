# Custom Items and Inventories with Scene Graph
Learn how to use Verse and Scene Graph to create custom items, customize the Fortnite player inventory, and more!
![Custom Items and Inventories with Scene Graph](https://dev.epicgames.com/community/api/documentation/image/39b7f0ee-a7ab-46fd-b8a2-15a718a70966?resizing_type=fill&width=1920&height=335)
The Custom Game Items and Inventories system is for designing and using items with unique attributes, functions and values in your game. Now, you can use Scene Graph and Verse to access the Custom Game Items and Inventories system in Fortnite. Using Scene Graph's entities and components, you can customize Fortnite's player inventory, and create custom items unique to your island.
