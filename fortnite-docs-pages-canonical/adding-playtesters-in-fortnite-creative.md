# Adding Playtesters
Set up playtesters to ensure the quality of your island gameplay.
![Adding Playtesters](https://dev.epicgames.com/community/api/documentation/image/7701be90-1e71-437a-bd38-efb954daf174?resizing_type=fill&width=1920&height=335)
Playtesters help to ensure island quality before publishing by working out the bugs and issues on your project through playtesting. This kind of quality control is part of a project release schedule and good practice for serious island developers.
Playtesters are people you invite to playtest your islands. They should be people you trust, and who will provide feedback on the quality of the experience and log any issues they encounter during gameplay.
Ideally, a playtester would look for possible issues with game mechanics, art, messaging, and clarity of goals in the game.
Playtesters do not have a hand in creating the island.
##  Create a Playtest
To turn the **Private Version** of your island into a playtestable project, follow the steps below.
  1. Open the **Projects** page and select **Playtests** from the tab menu.
  2. Click **Create Playtest** and a Private Version panel opens.
[![Playtests tab](https://dev.epicgames.com/community/api/documentation/image/f73b2f66-9175-410d-8dd0-55e3ae7f573b?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/f73b2f66-9175-410d-8dd0-55e3ae7f573b?resizing_type=fit)
  3. Select the **Private Version** from the panel and click **Create Playtest**.
[![Click Create Playtest](https://dev.epicgames.com/community/api/documentation/image/46375f67-261f-45bc-bd0a-4a6d7fa2efe4?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/46375f67-261f-45bc-bd0a-4a6d7fa2efe4?resizing_type=fit)

This creates a special code playtesters can use to access the island. The Private Version code is also visible from the private code message. Click **Manage Playtest Group** to add playtesters to the project.
[![A new playtest code is created for the project](https://dev.epicgames.com/community/api/documentation/image/586d85a5-043f-4acf-bba3-eb03a1d515c1?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/586d85a5-043f-4acf-bba3-eb03a1d515c1?resizing_type=fit)
Clicking the **Manage Team** button on the playtest island code takes you to the **Manage Teams** page where you can add playtesters to your project.
##  Adding Playtesters to a Project
Add playtesters to work in tandem with the project team. Follow the instructions below to invite playtesters to your team:
  1. Click **Manage Team** from the main menu.
  2. Click **Playtest Group** tab
  3. Click **Add New Playtesters** from the tabbed options.
[![Add playtester](https://dev.epicgames.com/community/api/documentation/image/eaf90711-4795-434f-a337-21e9bbc55339?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/eaf90711-4795-434f-a337-21e9bbc55339?resizing_type=fit)
  4. Click **Copy URL to Clipboard** to copy the invite URL.
[![Copy the invite URL](https://dev.epicgames.com/community/api/documentation/image/36ae10de-6fcf-4a31-b736-915f6b9531db?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/36ae10de-6fcf-4a31-b736-915f6b9531db?resizing_type=fit)
  5. Paste the URL into an email.
  6. Send the email to invite to people you trust to playtest your project.

The number of playtesters you can have is limited to 50. You’ll need to remove previous playtesters from the team if you want to add new playtesters.
When invitees accept the invitation, they are listed under the **Playtest Group** tab. Go into the Playtest Group tab to add playtesters to specific projects.
##  Accepting a Playtest Invitation
All playtesters receive an email invitation to join a team.
As a playtester, when you receive an invitation, click the link in the email to open the **Epic Games Sign In** page. If you don’t have an Epic Games account, you’ll be prompted to create one.
  1. Sign in to your Epic Games account.
[![Sign into Epic Games](https://dev.epicgames.com/community/api/documentation/image/c70a96ad-53e2-4d78-aa39-33a399b5e069?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/c70a96ad-53e2-4d78-aa39-33a399b5e069?resizing_type=fit)
  2. Click the **Join Playtest group** button. You’ll see a **Joined Successfully** message when you’ve joined the team.
[![Click Join Team button](https://dev.epicgames.com/community/api/documentation/image/06512c4b-f580-49ff-aab0-0b36eca489bb?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/06512c4b-f580-49ff-aab0-0b36eca489bb?resizing_type=fit)
  3. You must be an adult to accept a Playtest Invitation. If you haven’t done so already, you will need to complete the age verification process. Epic Games has partnered with SuperAwesome to confirm that you are an adult. The verification methods available to you will vary depending on where you live. See [Parent Verifications FAQs](https://parents.superawesome.com/parent-verification-faqs/) from SuperAwesome for more information.
  4. Click **Home** to go to the Creator Portal home page.
[![Click Home to go to your Creator Portal profile.](https://dev.epicgames.com/community/api/documentation/image/e0df5b5c-4f3d-4fcc-86a9-d33b67f73574?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/e0df5b5c-4f3d-4fcc-86a9-d33b67f73574?resizing_type=fit)

If the team you’ve been invited to join is at capacity for playtesters, you’ll receive a message:
[![If the project already has maximum number of playtesters, you’ll receive an error message](https://dev.epicgames.com/community/api/documentation/image/9b0ecc97-83a9-4962-80b9-9e49d3fe16c1?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/9b0ecc97-83a9-4962-80b9-9e49d3fe16c1?resizing_type=fit)
Contact the person who invited you to join the team.
##  Removing Playtesters
You can remove playtesters from a project once their testing is complete, or if you no longer need the person to playtest.
Go to the **Playtesters** tab under **Manage Team** to remove playtesters.
  1. Click the **ellipsis menu** next to the playtester’s name. The option to remove the playtester opens in a popup.
[![Select the ellipsis icon to remove a playtester](https://dev.epicgames.com/community/api/documentation/image/94f75048-b404-437b-96fa-8eef7ac34d31?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/94f75048-b404-437b-96fa-8eef7ac34d31?resizing_type=fit)
  2. Click **Remove Playtester** > **Confirm**. The person is removed from the Playtesters page.
[![Remove playtester.](https://dev.epicgames.com/community/api/documentation/image/ff7249b4-035b-41a9-b683-87501db417f4?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/ff7249b4-035b-41a9-b683-87501db417f4?resizing_type=fit)
You can add a new person after removing a playtester by clicking the **Add New Playtester** button.
[![Add a new playtester to a project](https://dev.epicgames.com/community/api/documentation/image/b6289a1e-41c5-4318-be1a-0ce6e7a19290?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/b6289a1e-41c5-4318-be1a-0ce6e7a19290?resizing_type=fit)

##  Switch Playtest Version
To playtest an iterated version of your island, update the playtest version and create a new playtest code.
  1. Open the project from the **Playtest** tab.
  2. Click **Update Playtest**.
[![Update Playtest button](https://dev.epicgames.com/community/api/documentation/image/bf0c3990-fa14-4815-9f4a-f30cf3701cd8?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/bf0c3990-fa14-4815-9f4a-f30cf3701cd8?resizing_type=fit)
  3. Select the project version from the slideout panel and click **Create Playtest**.
[![Select a new private version to turn into a plyatest project.](https://dev.epicgames.com/community/api/documentation/image/fb8a6dfd-49b2-4c32-80d2-31b6fc6708d3?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/fb8a6dfd-49b2-4c32-80d2-31b6fc6708d3?resizing_type=fit)

The island’s playtest code updates to a new code and shows the Private Version code for the newly selected version.
