# Animation and Cinematics
Animate skeletal meshs or create your own cinematics in UEFN! 
![Animation and Cinematics](https://dev.epicgames.com/community/api/documentation/image/1d1013cf-10cc-4fb9-aa43-232fe9c7df89?resizing_type=fill&width=1920&height=335)
Within UEFN you can import and assign animations to skeletal meshes. Then, using Sequencer, you can create a Level Sequence to make a cinematic that then trigger with the Cinematic Sequence Device.
